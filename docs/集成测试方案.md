# Fan MES 集成测试方案（前端操作版）

> 基于 `database/mysql/04_minimal_integration_test_data.sql` 联调测试数据设计
>
> 设计日期：2026-07-09
>
> **使用说明**：下文每个测试场景都从**浏览器前端页面操作**出发，描述"打开哪个页面 → 看到什么 → 点什么按钮 → 观察什么结果"。你需要先启动后端服务，然后用 `test_operator / 123456` 登录系统。

---

## 零、环境准备

### 1. 启动后端（必须）

```bash
cd fan-mes/backend
mvn clean package -DskipTests
# 按顺序启动各服务：gateway → auth → system → production → quality → equipment → report → integration
```

### 2. 导入测试数据（必须）

```bash
mysql -u root -p123456 < fan-mes/database/mysql/04_minimal_integration_test_data.sql
```

### 3. 启动前端

```bash
cd fan-mes/frontend
npm run dev
```

### 4. 登录

浏览器打开 `http://localhost:5173`，输入用户名 `test_operator`，密码 `123456`，点击登录。

> **登录后看到的主界面**：左侧是竖条导航栏（15 个图标），顶部栏显示"联调测试员"及登出按钮，中间是默认的"生产驾驶舱"仪表盘页面。
