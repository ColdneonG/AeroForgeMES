# 前端 UI 一致性与专业度审查报告

**审查日期**：2026-07-10
**审查范围**：`fan-mes/frontend/src/` 全部视图、组件、样式、布局、路由、配置
**审查性质**：只读审查，未做任何代码修改

---

## 一、总体结论

当前前端项目在 UI 层面存在**系统性的设计不一致问题**，核心矛盾在于项目中并存了**两套互不兼容的 CSS 设计体系**（`mes-*` 与 `siemens-*`），加上一套独立的看板/仪表板样式（`feature-*` / `board-*` / `plant-*` / `control-*`），导致用户在页面间导航时会感受到明显的视觉跳跃。

`theme.css` 定义了 36+ 个 CSS 变量，形成了一套完整的设计 Token 体系，但**这些变量在全局样式文件 `mes.css`（2226 行）中几乎完全未被使用**，所有颜色、间距、圆角、阴影均为硬编码值。这意味着设计意图与代码实现之间存在严重脱节。

从工业 MES 系统的专业度标准来看，当前前端存在以下核心风险：

1. **品牌一致性风险**：三套设计体系并存的现状使得系统难以呈现出统一、严谨的工业软件形象；
2. **可维护性风险**：`.api-state` 样式在 13 个文件中重复定义，任何全局样式变更都需要批量修改数十个文件；
3. **可扩展性风险**：缺乏统一的间距、字号、色彩规范，新增页面的样式质量完全依赖开发者个人判断；
4. **用户体验风险**：所有加载/错误/空状态仅为纯文本，没有骨架屏、加载动画或重试机制，不符合现代工业软件交互标准。

**正面方面**：CrudBoard 组件为 10 个业务页面提供了统一的结构框架；KpiCard 组件是唯一正确使用 CSS 变量的组件，可作为重构参考；整体配色方向（深蓝绿主色调 + 灰色背景）适合工业 MES 场景。

## 三、按模块列出问题

### 3.1 全局布局、侧边栏、顶部栏

- **侧边栏宽度定义冗余**：`siemens-shell` 定义 `--sidebar-width: 80px`（第 44 行），但 `mes-shell-expanded` 立即覆盖为 `76px`（第 255 行），基础值从未使用
- **模块侧边栏缺少关闭按钮**：用户只能通过点击外部区域或切换模块来关闭，没有 `×` 按钮或 Escape 键支持
- **侧边栏图标使用 SVG 图片**，但模块子菜单图标使用 Unicode 文本字符，两种图标渲染策略不一致
- **侧边栏深色背景（`#003b4a`）**与顶部栏（`#00799f`）之间缺乏视觉过渡，交界处略显生硬
- **logo 图片为 758KB 的 PNG**（`logo1.png`）和 947KB（`logo2.png`），文件过大，应使用 SVG 或压缩后的 WebP

### 3.2 登录页

- 整体布局合理，使用独立的 `.login-page` / `.login-card` 样式体系
- 错误颜色 `#bd2f2f` 与系统标准 `#b42318` 不一致
- 按钮高度 `40px` 与系统标准 `32px` 不一致
- 卡片圆角 `8px` 与系统卡片标准 `6px` 不一致
- 加载态仅改变按钮文字，无 spinner 或禁用样式动画

### 3.3 403 禁止访问页

- 使用 `font-size: 56px` 展示状态码，视觉突出但与其他页面无呼应
- "返回主页"链接样式化为按钮，行为正确

### 3.4 首页 / Dashboard（ManufacturingDashboard）

- 使用独立的 `.plant-dashboard` 样式体系，与标准页面布局完全不同
- 仪表盘使用 SVG 圆弧 + 指针，视觉质量较高但实现复杂
- 库存表格使用独立的 `.stock-card table` 样式（内边距 `5px 12px`），与系统标准表格（`11px 12px` 或 `7px 10px`）不一致
- 甜甜圈图使用了 3 种不同的绿色/青色（`#17a99e`、`#16a797`、`#16b3a2`），其中 `#16a797` 和 `#16b3a2` 肉眼几乎无法区分
- 9px 字体过小，在主流笔记本屏幕上可读性不足

### 3.5 生产管理

- **ProductionOrders**：使用 CrudBoard（mes-* 体系），整体结构规范。`mes-action-button` 按钮为实心填充风格

- **DispatchOrders**：使用 CrudBoard，与 ProductionOrders 一致
- **KittingBoard**：使用 CrudBoard，但缺料明细表使用了 Tailwind 调色板颜色（`#a0aec0`、`#1a202c`、`#e2e8f0`），并且自定义覆盖了 `.mes-table` 的内边距（`4px 6px` vs 全局 `11px 12px`），视觉上明显不一致

- **InspectionOrders**：使用 CrudBoard，结构规范

## 四、按问题类型归纳

### 4.1 字体与字号

- **严重缺乏字号规范**：使用 18+ 种不同字号（9px/10px/11px/12px/13px/14px/15px/16px/17px/18px/20px/22px/24px/28px/30px/32px/56px/64px），无固定类型比例
- **标题字号不统一**：CrudBoard h1=28px，Board h1=30px，Siemens h1=32px，仪表板 h1=32px
- **标题颜色不统一**：CrudBoard h1=#1f3139，Board h1=#25323b，Siemens h1=#4b5962
- **标题字重不统一**：Siemens h1 使用 `font-weight: 400`（较轻），其他页面使用默认粗体
- **非标准 `font-weight: 650`** 在 15+ 处使用，浏览器可能按 600 或 700 截断
- **Board 页面 h1 使用思源黑体**，CrudBoard 页面未指定字体族，继承 body 的 `"Segoe UI", "Microsoft YaHei"`
- **仪表板存在 `font-size: 9px`**，在 <1600px 屏幕上可读性不足
- **body 字体声明与 `--mes-font` 变量不一致**：body 使用 `"Segoe UI", "Microsoft YaHei", Arial, sans-serif`，变量使用 `"Microsoft YaHei", "PingFang SC", Arial, sans-serif`

### 4.2 颜色体系

- **主色偏移**：变量定义 `--mes-primary: #00759a`，实际使用 `#00799f`，差 4 个十六进制值
- **危险红碎片化**（5 种）：`#b23a3a`（语义色）、`#c01825`（告警三角形）、`#e01828`（库存危险）、`#ed1b2f`（圆弧/左边框）、`#bd2f2f`（登录错误）
- **成功绿碎片化**（6 种）：`#2e7d5b`（语义成功）、`#14b2a6`（进度条/强调）、`#17a99e`（甜甜圈默认）、`#16a797`（甜甜圈激活）、`#20aa9b`（迷你条形图）、`#65d6a5`（控制中心 OK）
- **背景灰碎片化**（7 种）：`#eeeeee`（body/仪表板）、`#eef2f4`（mes-workspace）、`#f6f8f9`（模块侧边栏）、`#f7fafb`（mes-table 表头）、`#f7f9fa`（feature-card 标题）、`#f2f4f6`（--mes-bg 变量，未使用）、`#e8eef2`（siemens-table 表头）
- **边框色碎片化**（6+ 种）：`#d7dde1`（最常用）、`#d8e1e5`（MES 面板）、`#d6e0e6`（--mes-line）、`#e2e8ec`（软边框）、`#b9c8d1`（强边框）、`#cbd8de`（输入框边框）
- **文本色碎片化**：至少有 7 种不同的深灰色/蓝灰色用于正文，5+ 种用于辅助文本
- **KittingBoard 引入 Tailwind 颜色**：`#a0aec0`、`#1a202c`、`#e2e8f0`，与 MES 调色板完全不匹配
- **控制中心深色主题完全硬编码**：颜色与亮色主题无逻辑映射关系
- **部分页面使用了渐变**：`sop-canvas` 使用 `linear-gradient(135deg, #f7fafb, #eaf4f6)`，在工业软件中略显花哨

### 4.3 间距与留白

- **无固定间距规范**：使用的间距值包括 2px/3px/4px/5px/6px/7px/8px/10px/12px/14px/16px/18px/20px/22px/24px/26px/28px/30px/34px/42px/48px/58px，无 4px 或 8px 基准体系
- **页面内边距不统一**：CrudBoard `22px 24px`，Board/Siemens `22px 26px 24px`，ControlCenter `20px 24px`
- **卡片内边距不统一**：mes-workspace 无直接内边距，siemens-panel-body `14px`，feature-card-title `0 14px`，line-card-top `8px 10px 0`
- **表格内边距不统一**：mes-table `11px 12px`，siemens-table `7px 10px`，feature-table `8px 10px`，stock-card table `5px 12px`
- **内容区间距（gap）不统一**：14px/16px/18px 混用

### 4.4 页面布局

- **三套页面容器并存**：`.mes-workspace`（overflow: auto）、`.siemens-page`（overflow: hidden, gap: 18px）、`.board-page`（overflow: auto），整体结构相似但细节不同
- **页面标题区结构不统一**：`.mes-page-heading`（flex, gap: 18px）vs `.siemens-page-header`（flex, gap: 24px, min-height: 54px）vs `.board-header`（flex, gap: 18px）vs `.dashboard-feature-header`（flex, gap: 20px）
- **筛选区不统一**：`.mes-filter-bar`（背景白, 圆角 6px, padding 12px）vs `.siemens-toolbar`（背景 #f7f9fa, 无圆角, padding 7px 10px）
- **面包屑（eyebrow）样式一致**但只在部分页面显示
- **信息密度差异大**：仪表板信息密度极高（4 列线卡 + 2 列仪表盘 + 库存表），DashboardFeaturePage 几乎为空，OperatorTerminal 有 6 列信息条 + 三栏操作面板

### 4.5 卡片样式

- **三种卡片/面板样式共存**：
  - `.mes-panel`：边框 `#d8e1e5`，圆角 6px，无阴影
  - `.siemens-panel`：边框 `#d7dde1`，无圆角，有阴影（`0 1px 2px rgba(26,48,60,0.1)`）
  - `.feature-card`：边框 `#d7dde1`，无圆角，有阴影（`0 1px 2px rgba(26,48,60,0.08)`）
